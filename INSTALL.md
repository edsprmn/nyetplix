## ğŸ›  Langkah 1: Aktifkan "Unknown Sources"
Secara default, Kodi memblokir instalasi dari luar. Kita perlu mengaktifkannya:
1. Jalankan Kodi, pilih ikon **Settings** (Gerigi) di pojok kiri atas.
2. Pilih menu **System**.
3. Pilih **Add-ons** di menu samping.
4. Aktifkan (Switch ON) pada bagian **Unknown sources**.
5. Klik **Yes** saat muncul peringatan.

---

## ğŸ“ Langkah 2: Tambahkan Alamat Repository
1. Kembali ke menu **Settings**, lalu pilih **File Manager**.
2. Klik dua kali pada **Add source**.
3. Klik pada tulisan **<None>**, lalu masukkan URL berikut dengan teliti:
   `https://edsprmn.github.io/nyetplix/`
4. Klik **OK**.
5. Di bagian nama (bawah), beri nama: `Nyetplix Repo`.
6. Klik **OK**.

---

## ğŸ“¦ Langkah 3: Instal Repository Nyetplix
1. Kembali ke menu utama Kodi, lalu pilih menu **Add-ons**.
2. Klik ikon **Box Terbuka** (Add-on Browser) di pojok kiri atas.
3. Pilih **Install from zip file**.
4. Cari dan klik nama yang tadi kita buat: `Nyetplix Repo`.
5. Masuk ke folder **zips**, lalu pilih **repository.nyetplix**.
6. Klik pada file zip yang tersedia (contoh: `repository.nyetplix-1.0.0.zip`).
7. Tunggu hingga muncul notifikasi *"Nyetplix Repository Add-on installed"* di pojok kanan atas.

## ğŸ¬ Langkah 4: Instal Add-on LK21 Official
1. Masih di menu yang sama, sekarang pilih **Install from repository**.
2. Pilih **Nyetplix Repository**.
3. Pilih **Video add-ons**.
4. Pilih **LK21 Official**.
5. Klik tombol **Install**.
6. Tunggu hingga muncul notifikasi *"LK21 Official Add-on installed"*.
